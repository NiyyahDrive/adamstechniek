# LazyDaysy

> Uw was, onze zorg. Vandaag nog schoon en fris terug.

Professionele wasseretteservice aan huis: ophalen, wassen, drogen, vouwen en dezelfde dag bezorgen.

## Stack
- Single-file HTML/CSS/JS, geen build-step
- Mobile-first, responsive (mobile / tablet / desktop)
- Fonts: Fraunces + Inter via Google Fonts
- Reveal-on-scroll via IntersectionObserver
- Respecteert `prefers-reduced-motion`

## Lokaal bekijken
Open `index.html` direct in de browser, of serveer met:
```bash
python3 -m http.server 8080
```

## Deploy
GitHub Pages: Settings → Pages → Source: `main` / root.
